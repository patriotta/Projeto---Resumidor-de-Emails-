# Resumidor Inteligente de Emails

Projeto em Python que utiliza IA para gerar resumos automáticos de emails, organizando as informações de forma clara e objetiva.

---

## Sobre o projeto

Este projeto simula uma caixa de entrada com diferentes tipos de emails (corporativo, marketing, suporte, etc.) e utiliza um modelo de inteligência artificial para gerar resumos curtos e diretos, mantendo a ideia principal de cada mensagem.

Os resumos são exibidos no terminal de forma organizada e também são salvos em um arquivo `.txt` para consulta posterior.

---

## Funcionalidades

-  Resumo automático de emails utilizando IA  
-  Tratamento de erros com tentativas de retry  
-  Organização dos dados em estruturas (listas e dicionários)  
-  Saída formatada no terminal  
-  Persistência dos dados em arquivo `.txt`  
-  Código modularizado (funções separadas por responsabilidade)  

---

## Tecnologias utilizadas

- Python  
- Google Generative AI (Gemini API)  
- Google Colab (ambiente de desenvolvimento)
